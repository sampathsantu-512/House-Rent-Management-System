# 🏡 HomeHaven - House Rent Management System

## 📌 Project Overview

HomeHaven is a full-stack House Rent Management System built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). It allows users to register, log in, browse rental properties, add listings, manage properties, book rentals, and view their bookings through a responsive web interface.

---

## 🚀 Features

### Authentication

* User Registration
* User Login
* JWT Authentication
* Protected Routes

### Property Management

* Add Property
* View All Properties
* Edit Property
* Delete Property
* View Property Details

### Search & Filters

* Search by Title
* Search by Location
* Filter by Property Type
* Filter by Bedrooms
* Filter by Price

### Booking System

* Book Property
* View My Bookings
* Cancel Booking

### Dashboard

* Total Properties
* Available Properties
* My Bookings

### User Interface

* Responsive Bootstrap Design
* Clean Navigation
* Mobile Friendly

## ⚙️ Installation

### Clone Repository

git clone <repository-url>

### Backend

cd server

npm install

Create a `.env` file using `.env.example`

npm start

### Frontend

cd client

npm install

npm start

---

## 🔐 Environment Variables

Create a `.env` file inside the **server** folder.

Example:

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

PORT=5000
